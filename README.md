## 推荐学习方法

### 第一遍：补齐知识

先读知识回顾，再跟做章内例题。遇到陌生结论时，回到参考书对应页补充理解，不要急着追求题量。

### 第二遍：独立刷题

给每道题设置时间限制。做题时先不看提示；超过建议用时仍无思路，再只看第一层提示，并重新独立完成。

### 第三遍：整理错题

不要只抄答案。建议记录三项：**卡在哪里、正确切入点是什么、下次看到什么特征应想到这个方法**。

### 考前冲刺：真题计时

通过真题索引定位历届试卷，按正式考试时长完成整套训练。完成后优先复盘不会做和“会做但丢分”的题。

公开网站不会要求你上传参考书。进入“原书阅读器”后，选择电脑上的 PDF 即可在浏览器本地阅读；章节中的页码链接会自动定位到相应页面。
