# 书籍式学习站设计

## 1. 设计结论

第二版放弃“仪表盘 + 卡片墙”，改为在线讲义：左侧全书目录、中间连续正文、右侧页边注。视觉上使用纸张底色、宋体/衬线正文、细分隔线和章节翻页，不使用大圆角、悬浮卡片和指标面板。

学习优先级固定为：

1. 知识点回顾；
2. 章内例题；
3. 独立练习与完整解析；
4. 原书考纲/真题页码；
5. 训练计划与进度。

计划退居附录，不占据首页核心位置。

## 2. 使用者与假设

- 使用者刚学完微积分 2，具备课内计算基础，但竞赛中的证明、方法选择、限时书写仍需强化。
- 默认参加 2026 年非数学专业 A 类初赛。
- 2026 年初赛时间按中国数学会第十八届通知确定为 11 月 14 日 09:00–11:30。
- 非数学专业 A 类初赛主线为高等数学；线性代数不进入当前初赛主线。

## 3. 信息架构

### 卷首

说明使用方法、剩余时间、本周书签和九章目录。

### 九章正文

每章统一采用：

1. 章节导语；
2. 参考指南考纲页码；
3. 4 个知识小节；
4. 必背公式；
5. 标准解题流程；
6. 高频失分点；
7. 章内例题；
8. 3 道练习；
9. 上一章 / 下一章。

### 附录

- 18 周训练表；
- 前十届非数学专业初赛试题与解答页码索引；
- PDF 原书阅读器；
- 补充资料站点；
- 18 条竞赛方法手册。

## 4. PDF 融合方式

参考书 PDF 共 382 个物理页面，约 90.35 MiB。

### 本地模式

PDF 保存在 `private/reference/`。Node 本地服务器以 Range 请求提供文件，PDF.js 直接在网页中渲染；章节页码链接如 `#/reference?page=26` 可准确跳转。

### 公开部署模式

公开站点不上传整本参考书。阅读器提供“选择本机 PDF”，文件由浏览器本地读取，不会上传服务器；选择后，所有章节和真题页码跳转仍然有效。

这样同时解决三件事：

- Cloudflare Pages 单个静态文件 25 MiB 限制无法容纳 90.35 MiB PDF；
- 避免公开分发受版权保护的整本图书；
- 保留用户要求的网页内阅读与知识点直达。

如后续需要跨设备私有阅读，可把 PDF 放到私有 R2，并用 Cloudflare Access 或短时签名链接控制访问，不应把 R2 公共桶直接暴露为公开下载。

## 5. 内容原则

- 知识回顾强调定理条件、概念层级和方法触发器，不只列公式。
- 例题与练习均为原创同型题，不复制参考书整题文本。
- 真题只做届次与原书页码索引，用户在个人副本中阅读。
- 每道题给出提示、完整解析、最终答案和易错点。
- 错题状态只记录“掌握 / 待回炉”，避免复杂游戏化。

## 6. 交互原则

- 解析使用教材式折叠段落，不使用弹窗。
- 目录使用章节列表，不使用图标导航。
- 进度只在页边注中轻量显示，不抢占正文。
- 公式由本地 KaTeX 渲染；PDF 由本地 PDF.js 渲染。
- 手机端隐藏右边注，将全书目录改为抽屉。

## 7. 成功标准

- 桌面与手机无横向溢出；
- 九章均有 8 个与本章主题对应的知识回顾小节；
- 81 道题全部可查看提示、完整解析、答案和易错点；
- 章节能跳转原书物理第 26–29 页；
- 真题索引能跳转第 92–144 页；
- 本地 PDF 第 26 页能在浏览器内实际渲染；
- 主站在 Render 后端不可用时仍可学习与刷题。
