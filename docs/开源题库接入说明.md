# 开源题库接入说明

## 接入结论

本轮把 GitHub 资源分为两类：许可证明确的内容可整理进站内题库；许可证缺失或来源权属不清的仓库只做人工调研，不复制题目。网页运行时不直接请求 GitHub，而是使用已经校对、翻译和适配现有数据结构的静态内容，避免外部服务不可用时影响学习。

接入后共有：

- 8 个许可证明确的 GitHub 来源；
- 18 个新增知识点，每章 2 个；
- 27 道开放拓展题，每章 3 道；
- 108 道站内题目，每章 1 道例题和 11 道练习。

## 改编边界

直接翻译改编的 OATutor 条目：

- `L10` 合并 `a342e92limit11` 与 `a342e92limit13`，把单侧无穷极限和竖直渐近线组成一道完整题；
- `I10` 基于 `a046eedintegrals1` 的指数原函数题，增加反常积分追问。

其余 25 道开放拓展题没有逐字复制仓库题目，而是依据各仓库公开的学习目标和主题结构重新编写为中文竞赛同型题。来源标注表示课程脉络和题型设计参考，不表示原仓库对本站内容背书。

## 未直接采用的仓库

调研过 `CUC-Life-Hack/further-mathematics-question-bank`。该仓库包含中文高等数学试卷整理，但 GitHub 页面没有许可证，且 README 说明题目主要来自校内往期试卷，因此不复制其题目、答案或 TeX 文件。后续只有在作者补充明确可复用许可证后，才重新评估是否接入。

## 数据与验证

- `public/open-bank-data.js`：来源、知识点和拓展题正文。
- `public/app.js`：在知识点和题目旁展示来源与许可证。
- `tests/content-test.cjs`：验证数量、章节归属、唯一 ID、来源和许可证。
- `tests/smoke-test.cjs`：验证桌面和手机渲染、来源显示与资料站链接。

更新开源内容时，先核验仓库当前许可证，再新增来源记录；不得把许可证为空的题目直接放进站内题库。
